# dotfiles

## Requirements

MacOS

## Install

1. Clone dotfiles repository
    ```shell
    git clone https://github.com/jimi024rion/dotfiles.git
    ```
2. Move to target directory
    ```shell
    cd dotfiles
    ```
3. Install
    ```shell
    ./install.sh
    ```
